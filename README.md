# Proyecto Módulo 1 – Fundamentos

## Descripción

Aplicación desarrollada en Streamlit que integra conceptos fundamentales de 
programación como variables, condicionales, estructuras de datos, funciones, 
programación funcional y POO. Contiene la clase principal app.py donde está toda 
la funcionalidad del código, requirements.txt con los requerimientos para 
ejecutar esta aplicación y el README con las instrucciones e indicaciones.

## Estructura del proyecto
- app.py 
- requirements.txt 
- README.md

## Funcionalidad
El proyecto usa un menpu desplegable para navegar entre los 5 apartados del proyecto

### Home
Carátula y presentación del proyecto
### Ejercicio 1
Calcula si un gasto entra dentro del presupuesto asignado.
### Ejercicio 2
Registra actividades como diccionarios dentro de una lista.
### Ejercicio 3
Calcula el retorno de las actividades usando una función, map y lambda.
### Ejercicio 4
Crea una Clase llamada Actividad conl a que convertir las actividades a objetos de esa clase.

## Instrucciones de Ejecución

1. Clonar el repositorio o descargar los archivos y descomprimir
2. Instalar dependencias (streamlit, la cual instala pandas también) teniendo instalado Python.
3. Ejecutar en la terminal el siguiente comando: streamlit run app.py
